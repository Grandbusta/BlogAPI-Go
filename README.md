# Blog API - Go

A simple Blog api built in Golang mainly for learning some concepts in Golang.
Very basic, more like a boilerplate to improve upon with time.

## Technologies Used

- Golang
- Gin framework
- MySQL

## Features

- Get list of all posts(/posts)
- Get a single post(/posts/:id)
